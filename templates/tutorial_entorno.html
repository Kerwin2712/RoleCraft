Misión: Configuración de Entorno

Antes de comenzar tu carrera en RoleCraft, debemos asegurar que tu estación de trabajo sea digna de un Experto. Sigue estos pasos para preparar tu entorno.

1. Visual Studio Code

El editor preferido de RoleCraft. Instálalo y domina los atajos básicos como Ctrl+Shift+P para la paleta de comandos.

Descargar VSCode

2. Python & Entornos Virtuales

Asegúrate de tener Python 3.10+ instalado. Siempre trabaja en un entorno virtual aislado:

python -m venv env
# Windows: .\env\Scripts\activate

3. Extensión Antigravity

Esta extensión es tu compañera de IA. Actívala y dale acceso al directorio de tu proyecto para que pueda entender tu código en tiempo real.
